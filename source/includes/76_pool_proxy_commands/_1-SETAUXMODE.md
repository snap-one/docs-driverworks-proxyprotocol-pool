## SET\_AUX\_MODE

Received from the proxy when the auxiliary mode has been changed.


### Name

`SET_AUX_MODE ()`


| Parameter | Type | Description                       |
| --------- | ---- | --------------------------------- |
| `ID`      | NUM  | ID value of the auxiliary to set. |
| `MODE`    | STR  | ON/OFF                            |


### Returns

`None`
