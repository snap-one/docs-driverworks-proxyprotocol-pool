## SET\_POOL\_HEAT\_MODE

Received from the proxy when pool het mode has been changed.


### Name

`SET_POOL_HEATMODE ()`


| Parameter | Description | Description        |
| --------- | ----------- | ------------------ |
| `MODE`    | STR         | ON/OFF             |
| `ID`      | INT         | Heat mode ID value |


### Returns

`None`
