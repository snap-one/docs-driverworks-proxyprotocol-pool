## POOL VARIABLES

The Pool Proxy supports the following variables:

`AIR_TEMPERATURE` integer


`POOL_HEATMODE` string

`POOL_SETPOINT` integer


`POOL_TEMPERATURE` integer


`PUMPMODE` string


`SCALE` string


`SPAMODE` string


`SPA_HEATMODE` string


`SPA_SETPOINT` integer


`SPA_TEMPERATURE` integer
